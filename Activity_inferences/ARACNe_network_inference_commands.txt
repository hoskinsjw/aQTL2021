### These are the commands for running the bootstrap ARACNe network inferences for the TwinsUK subcutaneous adipose RNA-seq data. Note that the aracne.jar file must be in the working directory,
### which can be downloaded from the Califano lab GitHub here: https://github.com/califano-lab/ARACNe-AP

# First, calculate mutual information threshold
java -Xmx8G -jar Aracne.jar -e Eurobats_adipose_expressed_genes_logTPM.txt -o ARACNEoutput --tfs Full_Califano_regulator_list_plus_LINC-PINT_by_symbol.txt --pvalue 1E-8 --seed 1 --calculateThreshold
